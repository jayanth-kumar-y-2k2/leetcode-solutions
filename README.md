# 150 LeetCode Solutions

## Overview

This repository contains curated, concept-wise solutions for the essential 150 LeetCode problems. Each solution is organized by topic so you can study algorithms and patterns efficiently.

## Repository Structure

- `/<topic>/<problem-number>_<slug>.py`: Solutions grouped by topic (for example `arrays/001_two_sum.py`).
- `README.md`: Project overview and how to contribute.
